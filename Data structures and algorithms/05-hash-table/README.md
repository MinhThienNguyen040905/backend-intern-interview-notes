# 05 - Hash Table, Hash Map và Hash Set

## Mục tiêu

- Dùng hash map để đếm, lưu index, nhóm dữ liệu.
- Dùng hash set để kiểm tra tồn tại.
- Hiểu average `O(1)` và collision.
