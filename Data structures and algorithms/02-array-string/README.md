# 02 - Array và String

## Mục tiêu

- Nắm thao tác cơ bản với array và string.
- Biết các pattern phổ biến: two pointers, sliding window, prefix sum.
- Xử lý edge cases: rỗng, một phần tử, duplicate, chữ hoa/thường.

## File trong chủ đề

- [KNOWLEDGE.md](./KNOWLEDGE.md): kiến thức và dạng bài.
- [src/Main.java](./src/Main.java): ví dụ reverse string và maximum subarray.
