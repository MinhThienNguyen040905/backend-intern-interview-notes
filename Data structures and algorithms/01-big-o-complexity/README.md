# 01 - Big O và phân tích độ phức tạp

## Mục tiêu

- Hiểu `time complexity` và `space complexity`.
- Biết rút gọn Big O khi đọc code.
- Phân tích được vòng lặp, recursion và cấu trúc dữ liệu thường gặp.

## File trong chủ đề

- [KNOWLEDGE.md](./KNOWLEDGE.md): kiến thức chi tiết và lỗi hay gặp.
- [src/Main.java](./src/Main.java): ví dụ đo số bước tăng theo input.

## Bài nên luyện

- Phân tích complexity của vòng lặp đơn.
- Phân tích complexity của vòng lặp lồng nhau.
- Phân tích binary search.
- Phân tích recursion Fibonacci.
