# 13 - Graph

## Mục tiêu

- Biểu diễn graph bằng adjacency list/matrix.
- Dùng DFS/BFS cho connected components và shortest path không trọng số.
- Biết topological sort, Dijkstra ở mức phỏng vấn.

## File trong chủ đề

- [KNOWLEDGE.md](./KNOWLEDGE.md): thuật toán graph chính.
- [src/Main.java](./src/Main.java): ví dụ BFS adjacency list.

## Bài nên luyện

- Number of Islands.
- Clone Graph.
- Course Schedule.
- Rotting Oranges.
- Network Delay Time.
